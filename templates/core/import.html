Transactional Rollback Safeguards

Both CSV and JSON imports are executed inside strict atomic database transactions. If a single validation mismatch, bad cell, or missing foreign key is encountered, the entire import rolls back instantly to keep your data 100% clean.

CSV / Excel Table Ingestion

Select a master data table and upload a CSV file. If the file contains an id column with matching IDs, those records will be updated; otherwise, new records are created.

{% csrf_token %}

CSV Formatting Guidelines:

  • Column headers must match the database field names exactly (case-insensitive).
  • For relational fields (e.g. default_expense_account or default_revenue_account), enter the parent record's ID (e.g., Account Code like 4100).
  • To find out the correct column headers, export the table using our Export tool first!
